what is difference between <header> and <footer> tag?

Can we use header inside <footer> tag</footer>?: 
Using <header> inside <footer>:
Technically, HTML allows nesting elements, but it's semantically incorrect to 
include a <header> inside a <footer>. Headers and footers are distinct sections of a 
document, and they serve different purposes. Nesting them this way would violate the 
intended structure and meaning of the HTML document.

Difference between header and h1 tag?
---- The <header> tag is used to define a header for a document or a section. It typically 
contains introductory content like logos, navigation menus, or headings.
It's not limited to containing just a single heading, and it can include various 
elements to structure the header of a webpage.
---- The <h1> tag is a heading tag used to define the most important heading on a page or in a section.
It's typically used to represent the main topic or title of the content.
